Escaped RDFa

An XML namespace extension enabling RDFa embedding in restrictive HTML environments

Escaped RDFa (eRDFa) provides a mechanism for escaping RDFa content so it can be preserved in constrained publishing platforms like blogging systems that strip RDFa attributes. It defines two key terms: 'example' for illustrative markup and 'embedded' for escaped RDFa that should be processed.
